为文章添加图片

نُشر في 2020-04-22

咳咳同资们，开始上课了

​ 在此之前，如果看过前几篇博文的同志们，相信一点能够发现我的博客里出现的一些小问题。大家刚看到我的博文时，看到的默认为页面为左侧导航栏中（ “Startseite” 首页）的内容，这时几篇博文的文字和图片都是可以正常显示的，而如果我们要想通过（ “Archiv“ 目录）来查看博文时，却看不到图片。嘿嘿，这里要告诉大家，由于本人还是个小白，之前刚操作时出现了一些技术上的问题，那么从这边博文开始，本人决定痛改前非，重新做文章，教你们正确的方法！所以，，，前面的文章你们就先在首页里观看啦！感谢！

经过测试，发现如下方法有效：

1、首先把blog目录下的_config.yml里的psot_asset_folder:设置为true

2、在blog目录下执行git命令: npm install hexo-asset-image –save

3、在git bash here的窗口中执行hexo n “文章名”，此时在post文件夹下看见与文章名同名的文件，且typora自动打开
